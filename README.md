Student Profile Score Calculator

A multi-step form that calculates how competitive your academic profile is for studying abroad. Scores your GPA, IELTS, financial proof, and experience out of 100 with live updates.

## Features
- 2-step form with validation before proceeding
- Live score preview as you type
- Animated score breakdown with progress bars
- Band labels: Strong / Moderate / Weak / Not Viable
- Same scoring logic used in the StudyAI backend

## Tech
React + TypeScript + Vite. No external dependencies.

## Run locally
```bash
npm install
npm run dev
```
